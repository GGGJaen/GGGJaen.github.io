Nubes de puntos

Este documento explica algunos aspectos de la clase GEU::PointCloud. Esta clase encapsula las utilidades asociadas al manejo de una nube de puntos.

:blue_book: Instrucciones

Como cargar la nube

Para cargar la nube utilizaremos el FileManager. Mas concretamente, la función loadPointCloud. Dado que FileManager es una clase estática, la llamada a esta función se realizara de la siguiente manera: FileManager::loadPointCloud(...). Cabe destacar que actualmente esta función solo soporta la carga de ficheros PLY. Si se desea cargar un LAS o similar, se deberá de convertir previamente a PLY (Cloud Compare es un buen amigo para esto).

En el caso de que queramos cargar todas las nubes que hay en un directorio, podemos usar la función loadAllCloudsUnderFolder(folder). Ambas funciones están especificadas con Doxygen, por lo que el IDE te puede dar mas información sobre como funcionan exactamente estas funciones.

Como obtener la nube

Para obtener la nube se debe utilizar la función getPointCloud de la clase Renderer. Esta pedirá como único parámetro el nombre de la nube. Nos devolverá un puntero a la misma en el caso de que esta exista en la escena activa o nullptr en caso contrario.

const auto cloud = GEU::Renderer::getInstance()->getPointCloud("ExamplePointCloud");
if(cloud){
//Do things
}

Como añadir puntos

Se pueden utilizar los métodos addNewPoint, addNewPoints o obtener el vector de puntos mediante getPoints y modificarlo directamente. Todas estas modificaciones se pueden realizar desde hilos distintos al principal, ya que todas las llamadas a OpenGL se encuentran aisladas de estos métodos.

Si se desea añadir un gran número de puntos, es recomendable usar el método addNewPoints en vez de múltiples llamadas a addNewPoint.

Como modificar los puntos de la nube

Se pueden obtener los puntos mediante getPoints. Una vez se hayan terminado de modificar, se deberá de llamar al método needUpdate para que se refleje la actualización en el renderizado.

const auto cloud = GEU::Renderer::getInstance()->getPointCloud("ExamplePointCloud");
if(cloud){
const auto& points = cloud->getPoints();
//Do things with the points.
cloud->needUpdate();
}
