void GEU::ProceduralGenerator::computeNURBS ( GEU::PointCloud * pointCloud,
unsigned degree,
unsigned divX,
unsigned divY,
float desiredDensityMultiplier,
bool variateColors = false )

Method to generate the NURBS based point cloud for the terrain. The purpose of this method is to increase the ground cloud definition while maintaining this cloud characteristics (Curvatures, points colors...)

Parámetros
degreeNURBS degree
divXnumber of X subdivisions used by statitics method
divYnumber of Y subdivisions used by statitics method
desiredDensityMultiplierDensity multiplier used to generate the NURBS point cloud. Using a value of 1, you will generate points only in voxels that don't have enought points (Empty voxels or voxels with big holes for example)
