Glosario de términos

En esta sección se describe terminología utilizada en el entorno de GEU o del grupo GGGJ y que es necesaria para poder adaptarse rápidamente al entorno de desarrollo de GEU.

  • Meshlet: Agrupación de varios puntos espacialmente cercanos entre si. El objetivo de estas entidades es la de acelerar ciertos procesos mediante la inferencia de información relativas a puntos de la unidad.
    • Ejemplo: Si un meshlet es visible por su caja envolvente entonces todos los puntos que lo componen también lo serán, mientras que si el meshlet no es visible te ahorras la comprobación para todos los puntos que contenidos en el mismo.
  • Multiespectral: Referido a la captura de datos dentro de unos rangos de longitud de onda específicos a través del espectro electromagnético
    • Un sensor o cámara multiespectral será capaz de capturar imágenes en diferentes bandas del espectro de luz, más allá de los que nuestros ojos pueden ver.
    • Generalmente al hablar de multiespectral nos referimos a la captura de 4 o 5 bandas que suelen ser:
      • Blue (450-500 nm)
      • Green (500-570 nm)
      • Red (620-690 nm)
      • Infrarroja cercana o NIR (760-900 nm)
      • Borde rojo o REG (680-750 nm)
  • Hiperespectral: Referido a la captura de datos dentro de unos rangos de longitud de onda específicos a través del espectro electromagnético. Al contrario que la Multiespectral en la Hiperespectral se recorre todo el espectro electromagnético
    • Un sensor o cámara hiperespectral será capaz de capturar información de cientos de bandas del espectro de luz
  • Hypercube: Referido a la estructura de datos capaz de almacenar toda la información hiperespectral ligada a una imagen. Se desarrolla como un cubo ya que se parte de una imagen 2D (como una matriz) y se van añadiendo tantas filas como bandas hiperespectrales se procesen.
  • Servidor NAS: Dispositivo de almacenamiento conectado a la red. Su funcionalidad se encuentra orientada hacia el almacenamiento de datos de gran escala y tamaño
  • Ortorrectificar: Proceso fotogramétrico que asegura que los elementos o pixeles de una imagen se encuentran en su respectiva posición geográfica. Consiste en la eliminación de distorsiones o desplazamientos de la imagen causados por la inclinación del sensor y el relieve topográfico.
  • LiDAR (Ligth Detection and Ranging): Dispositivo que permite determinar la distancia desde un emisor laser a un objeto o superficie mediante un haz láser. Mediante el uso de estos dispositivos es posible generar nubes de puntos del terreno.
  • SFM (Structure for motion): Es el proceso de generar una estructrura en 3D a partir de un conjunto de imagenes 2D tomadas desde diferentes puntos de vista de la escena
  • Índices de vegetación: Combinaciones de bandas espectrales cuyo objetivo es realzar la cubierta vegetal en función de la respuesta espectral de la misma. Se buscaran aquellas combinaciones que resultan la vegetación y atenúen otros elementos como agua, suelo, edificios... Algunos ejemplos de indices de vegetación son:
    • NDVI (Normalized Difference Vegetation Index) Es el índice de vegetación por excelencia y algunos de los desarrollados a posteriori buscan resolver algunas de sus carencias como son la sensibilidad a los efectos atmosféricos y del suelo.
    • SAVI (Soil Adjusted Vegetation Index) Fue diseñado para minimizar las influencias del birllo del suelo, se calcula aplicando un factor de ajuste a la ecución del NDVI
    • EVI (Enhanced Vegetation Index) Busca expresar los efectos atmosféricos calculando la diferencia de radiancia entre las bandas azul y rojo
  • DJI: Marca de cámaras, sensores y drones con los que se realizan vuelos para la obtención de datos multiespectrales (DJI 3M), térmicos (DJI 3T) y rgb.
  • Parrot: Modelo de sensor multiespectral implementado en GEU y con el que se puede generar la fusión de datos sobre nubes de puntos RGB.
  • Micasense: Modelo de sensor multiespectral implementado en GEU y con el que se puede generar la fusión de datos sobre nubes de puntos RGB.
  • Fusión de datos: Proceso de integración de datos multifuente (térmicos, multiespectrales o hiperespectrales) sobre una nube de puntos. En este proceso para cada punto de la nube se asignan sus valores para cada atributo fusionado siendo para la térmica un valor y para la multiespectral e hiperespectral un valor por banda del espectro. Además se pueden añadir mas atributos a cada punto como pueden ser índices de vegetación
  • Ráster: Fichero de datos en forma de rejilla rectangular que representan valores para posiciones del mundo real. Un ejemplo de Ráster son los DTM o DEM.
  • DEM (Digital Elevation Model) : Representación ráster de una superficie continua. En este caso en el ráster la matriz almacena los valores de elevación del terreno para cada coordenadas del mundo ubicada en cada pixel.
  • DTM: Representación ráster de una superficie continua. Es una ampliación de un DEM donde en este caso el ráster incluye elementos geográficos como ríos o crestas.
  • DSM (Digital Surface Model) : Representación ráster de una superficie continua. Es el último nivel de representación donde se incluye vegetación, edificaciones y objetos presentes además del terreno.
  • Reflectancia: Es la propiedad de un cuerpo o material para reflejar la luz. En el campo de la captación y fusión de datos la reflectancia nos permite dar valores a cada elemento del terreno al usar un sensor multiespectral o hiperespectral y a posteriori tratar de diferenciarlos mediante estas propiedades.
  • GIS (Geographical Information System) : Conjunto de herramientas que permite el almacenamiento, manipulación y análisis de datos de origen espacial procedentes del mundo real.
  • Nube puntos: Formato de representación de un modelo 3D mediante el uso de vértices con coordenadas espaciales X,Y,Z. Sobre estos vértices además se pueden aplicar propiedades como la del color para aplicar realismo o gradientes para observar el cambio de ciertas propiedades en el espacio.
  • Signatura espectral: Tambien conocido como firma espectral se refiere al perfil específico de radiancia de un elemento a lo largo de todas las longitudes de onda del espectro.
