◆ loadPointCloud()

bool GEU::FileManager::loadPointCloud ( const std::string & filepath,
bool useClassification = true,
bool optimizeCloud = true,
bool asyncOptimize = true )
static

Carga la nube de puntos especificada.

Parameters
filepathruta a la nube de puntos. Debe de contener la extension
useClassificationSi es true y la nube tiene datos de clasificacion, los cargara. En caso de ser false, no se cargarán.
optimizeCloudSi es true, se realizara una ordenacion espacial de la nube, lo que mejora la eficiencia de renderizado de la misma. Si es false, no se realizara, perjudicando el rendimiento del renderizado pero favoreciendo el tiempo de carga.
asyncOptimizeSi es true, el proceso de optimizacion se realizara en un hilo secundario, haciendo que el tiempo de carga total sea inferior.
Returns
bool dependiendo de si se ha cargado correctamente o no.
