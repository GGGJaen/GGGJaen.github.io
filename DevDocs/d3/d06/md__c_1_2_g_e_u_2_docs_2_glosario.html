Glosario de herramientas

En esta sección se enumeran las herramientas que son de utilidad para poder trabajar en el entorno de desarrollo de GEU. Para cada herramienta se dará una breve descripción, en caso de querer mas información se podrá obtener en su enlace asociado o en la sección Discussions del repositorio de GitHub.

  • GEU
    • Aplicación gráfica interactiva, desarrollada para equipos de escritorio con el sistema operativo Windows y tarjetas gráficas Nvidia.
    • La aplicación está diseñada para tratar, principalmente, con nubes de puntos y su interacción con diversas fuentes de información
  • Premake5
    • Utilidad de desarrollo para la creación automática de configuraciones de proyectos a partir de su código fuente
    • Trabaja con lenguajes como C, C++ y C#
    • En el entorno de GEU se utiliza para la generación del proyecto en Visual Studio
  • Visual Studio Community 2022
    • Entorno de desarrollo integrado para Windows compatible con el lenguaje C++
    • Es el entorno elegido para el desarrollo de GEU y todo el proceso de instalación y generación del proyecto esta orientado a su uso
  • VCPKG
    • Instalador de paquetes en línea de comandos para C++.
    • Facilita la instalación inicial del Software GEU debido a la gran cantidad de dependencias con las que cuenta el programa.
  • GitHub Desktop
    • Software para control de versiones.
    • Permite gestionar proyectos de gran dimensionalidad de forma fácil gracias a su interfaz gráfica.
  • CloudCompare
    • Software para la visualización y procesamiento de nubes de puntos 3D.
    • Permite trabajar con los datos de forma previa a su inserción en GEU si fuera necesario.
  • Spectral View
    • Software para corrección y procesamiento de líneas hiperespectrales.
    • Permite ortorrectificar los datos hiperespectrales para poder trabajar con ellos en GEU.
